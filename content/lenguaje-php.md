# Estadísticas

> [!important] Estadísticas de utilización
> PHP es el lenguaje con mayor utilización en sitios web con computación en el servidor.

> [!imporant] Estadísticas de versiones utilizadas
> Las versiones 8.1 y 8.2 están en la zona naranja, con lo cual solo tienen cambios con respecto a mejoras críticas de seguridad, así que a excepción de eso no va a tener ningún tipo de soporte.
> Y las versiones que están en rojo (7.4 y 8.0) ya no tienen ningún tipo de soporte.

# Fundamentos del lenguaje
## Introducción

> [!note] Código abierto
> PHP es un lenguaje de programación de código abierto para el desarrollo web.

> [!note] Multiplataforma
> Prácticamente disponible para todos los sistemas operativos.

> [!important] PHP soporta todos los paradigmas
> Es posible mezclar paradigmas.

> [!quote] Laraval es el framework de PHP más utilizado.

> [!note] Código PHP
> ```php
> <?php
> 	instruccion1,
> 	instruccion2
> 	...
> 	instruccionN
> ?>
> ```

> [!note] Hola mundo
> ```php
> <!DOCTYPE html>
> <html>
> </head>
> 	<title> Hola </title> 
> </head>
> </body>
> 	<h1> Hola.php </h1>
> 	<?php echo "<p> ¡Hola mundo! </p>"; ?> 
> </body>
> </html>
> ```
